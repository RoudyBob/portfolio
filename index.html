A Little Bit About Me

                                ★                                

I have worked in IT for over 24 years, most recently focusing on Cloud Computing platforms such as Google Cloud Platform and Microsoft Azure. I've always been fascinated with software developement. I can remember typing in BASIC programs that I found in magazines into my very first computer - a XT 286-compatible personal computer that my dad bought for the family.

I've seen many big technological transitions during my career. The current shift to cloud computing and the rise of cloud-native applciations being the most recent and perhaps the most significant. As a developer, knowing how to leverage the code you write to take advantage of all of these powerful platforms and capabilities is key. With just some well-written code and the power of huge cloud platforms like Google Cloud, AWS and Azure, the possibilities are limitless. It was for this reason that I decided to spend time really focusing on learning how to write software and hone my development skills.
